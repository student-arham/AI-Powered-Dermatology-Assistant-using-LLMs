# AI-Powered-Dermatology-Assistant-Using-LLMs

Using OpenCV, YOLOv8, Streamlit, Groq API, ReportLab, Flutter

 Implemented computer vision techniques (OpenCV) to analyze skin images and classify conditions like acne,
pigmentation, and dryness.

• Integrated LLM (via Groq API) to generate detailed medical insights including symptoms, causes, and treatment
suggestions.

• Designed an interactive UI using Streamlit with real-time analysis, confidence visualization, and patient history
tracking.

• Automated PDF report generation using ReportLab for downloadable medical summaries
